PARA EJECUTAR EL MEMORAMA, SE TIENE QUE PONER EN LA TERMINAL python3 main.py

El memorama consiste de dos menus con los que podras moverte dentro del programa, el primer menu, sirve para registrarse o entrar como un usuario ya registrado, al estar ya en tu cuenta, se abre un segundo menu, el cual funciona para poder jugar, ver tus estadisticas, borrar esas estadisticas restableciendo su valor a 0, por ultimo una opcion para salir, la forma de jugar es dando dos numeros "a# y "b" escribiendolos de la sigueinte forma <"a" "b">, el programa no se rompe si se ingresan numeros invalido. Aun asi no lo protegi de que se le ingrese un valor no numerico.
